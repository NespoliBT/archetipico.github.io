Visiting this website you're subjected to GitHub Pages privacy policies.

This website uses web storage to enhance its functionalities. Of course no sensitive data is stored on the web storage, but rather things like:

  • sessionStorage["device"]: to understand what device you're using, so the page can resize
  • localStorage["mode"]: to remember if you are a dark mode or light mode user
